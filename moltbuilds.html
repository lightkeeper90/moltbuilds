🎉 Configuration Compiled Successfully

Your moltbuilds-alpha agent is ready. Follow the installation steps below.

1
Install Moltbuilds CLI
bash
npm install -g @moltbuilds/cli@latest
2
Initialize Agent Project
bash
moltbuilds init moltbuilds-alpha --template=agent-v2 --model=gpt-4o

💡 Pro Tip

Add --tools=browser,calculator to pre-configure tools during init.

3
Install & Build
bash
cd moltbuilds-alpha && npm install && npm run build
4
Start Agent
bash
npm run start

🚀 Production Deployment

  • Use pm2 start npm --name "agent" -- run start for 24/7 operation
  • Run moltbuilds doctor to verify configuration
  • Set LOG_LEVEL=debug for troubleshooting
